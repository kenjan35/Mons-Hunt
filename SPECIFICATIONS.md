# Mons-Hunt

1. Presentation of the project
### Project's name
**Mons-Hunt**

### Description
Mons-Hunt is a solo/multi action/RPG video game in which the player embodies a hunter tasked with tracking and defeating monsters in different areas.
The gameplay loop is:

- Accept a quest
- Hunt monsters
- Collect loot
- Craft better gear
- Fight stronger monsters

2. Project Goals
Main Goals
- Create a simple monster hunting game
- Make combat fun and responsive
- Add player progression through equipment upgrades
